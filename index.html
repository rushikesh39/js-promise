Q1:-Write one example explaining how you can write a callback function. 

        function add(num1, num2, call){
            let sum=num1+num2;
            call(sum);
          }
          function result(show){
            console.log(`The result is: ${show}`);
          
          }
          add(12,2,result);

The given code defines two functions add and result, and then calls add with the arguments 12, 2, and the function result as the callback. The add function adds num1 and num2 to get a sum, and then calls the call function, passing in the sum as an argument. The result function simply displays a message with the show.

So when the add function is called with the arguments 12, 2, and result, it will calculate the sum 14, and then invoke the result function with the show parameter set to 14. The result function will then log the message "The result is: 14" to the console.

        function myFunction(name, callback) {
            console.log('Hi' + ' ' + name);
            callback();
          }
          
          function callMe() {
            console.log('I am callback function');
          }
          myFunction('rushi', callMe);

The myFunction function takes a name argument and a callback function argument. It logs a message to the console using the name argument, and then calls the callback function. The callMe function is passed as the callback argument to myFunction. When myFunction is called with the name 'rushi', it logs the message "Hi rushi" to the console and then calls the callMe function, which logs the message "I am callback function" to the console.

Q6:- Create examples to explain callback hell function

      function hellFunc(alpha, dealy, callBack){
        setTimeout(() => {
            console.log(alpha);
            callBack()
        },dealy);
      }
      function callHell(){
        hellFunc("R", 2000, () => {
            hellFunc("U", 3000, ()=> {
                hellFunc("S", 1000, ()=> {
                    hellFunc("H", 6000, ()=> {
                        hellFunc("I", 5000,()=>{})
                    })
                })
            })
        })
      }
      callHell();

The function callHell invokes the hellFunc function multiple times with different values of alpha, each with a different dealy before invoking the next function in the chain as the callBack. Overall, this creates a sequence of function calls with specified intervals in between, allowing the code to implement a delay and execute code in the correct sequence.

Q9:-Create examples to explain promise.all function

      const promise1 = new Promise((resolve, reject) => {
        setTimeout(() => {
          resolve('Promise 1 resolved');
        }, 10000);
      });
      
      const promise2 = new Promise((resolve, reject) => {
        setTimeout(() => {
          resolve('Promise 2 resolved');
        }, 11000);
      });
      
      const promise3 = new Promise((resolve, reject) => {
        setTimeout(() => {
          resolve('Promise 3 resolved');
        },12000);
      });
      
      Promise.all([promise1, promise2, promise3])
        .then((response) => {
          console.log(response);
        })
        .catch((error) => {
          console.error(error);
        });

The Promise.all() method takes an array of Promises as its input. In this case, it takes an array containing promise1, promise2, and promise3. The .then() method is called on the returned Promise to handle the resolved data.
